# Academic Event Registration Portal

This is a web application that allows students to register for academic events.

## Features
- Student Registration
- Student Login
- View Events
- Register for Events
- View My Registrations

## Technologies Used
- PHP
- MySQL
- HTML
- XAMPP

## How to Run
1. Install XAMPP
2. Place project inside htdocs
3. Start Apache and MySQL
4. Import database in phpMyAdmin
5. Run: http://localhost/academic_event_portal/
